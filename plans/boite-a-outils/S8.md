## But

Permettre au soignant d'adapter la réserve d'activités au public : un interrupteur on/off qui
masque les activités d'intensité légère pour ne montrer que les modérées (et intenses si cette
catégorie apparaît un jour).

## Conception (fixée)

1. State local `toniquesUniquement: boolean` (défaut `false` — comportement actuel intact).
2. Interrupteur au-dessus de la réserve d'activités : un bouton bascule (`role="switch"`,
   `aria-checked`) libellé « Activités toniques uniquement », avec état visuel on/off à double
   encodage (position + libellé « Toutes / Modérées+ » — pas de couleur seule).
3. Filtre : quand actif, la **réserve** n'affiche que `intensite !== 'légère'`.
4. **Les activités déjà placées dans la jauge restent en place** même si le filtre les aurait
   masquées (on ne retire jamais un choix du patient) — le filtre ne s'applique qu'à la réserve.
5. Cible ≥ 44 px, libellé lisible à 1 m, aucune persistance.

## Étapes

1. State + interrupteur → 2. Filtre réserve → 3. Gate : `npx tsc --noEmit` + `npm run build`
