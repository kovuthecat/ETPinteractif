## AP2 — `SilhouetteCorps` générique, silhouette diabète en wrapper

### Objectif

Rendre la silhouette réutilisable par n'importe quel thème (conformément à l'invariant
multi-thèmes : le moteur ne connaît aucun thème par son nom), sans rien changer pour les 9 modules
diabète.

### Décisions clés

- **Créer `src/components/SilhouetteCorps.tsx` + `SilhouetteCorps.module.css`** en déplaçant le code
  actuel de la silhouette diabète, avec cette API générique :

  ```ts
  export type SilhouetteEtat = 'actif' | 'ouvert' | 'verrouille' | 'allume' | 'masque';
  export interface SilhouetteZone {
    id: string;            // libre — le thème choisit ses ids
    label: string;
    x: number; y: number; r: number;   // ancre dans le repère SILHOUETTE_VIEWBOX
    etat: SilhouetteEtat;
  }
  export const SILHOUETTE_VIEWBOX = { width: 340, height: 760 };
  export interface SilhouetteCorpsProps {
    zones: SilhouetteZone[];
    onZoneClick?: (id: string) => void;
    /** Motif central discret (le trait « vaisseaux » actuel) — optionnel, activé par défaut
     *  pour ne rien changer côté diabète. */
    motifVaisseaux?: boolean;
    children?: ReactNode;   // annotations positionnées par le consommateur
  }
  ```

- Le dessin du corps, la technique pastille = vrai `<button>` HTML ≥ 44 px, les 5 états et leurs
  icônes (Lock / CheckCircle2 / ShieldCheck), les labels sous pastille, le CSS : **déplacés tels
  quels** (mêmes classes, mêmes valeurs). Seule différence : zones/ancres viennent des props au lieu
  des tables locales.
- **`src/features/diabete/components/Silhouette.tsx` devient un wrapper fin** qui :
  - conserve **exactement** ses exports actuels (`ZoneId`, `ZoneEtat`, `SilhouetteZoneState`,
    `SilhouetteProps`, `SILHOUETTE_VIEWBOX`, `SILHOUETTE_ANCHORS`, défaut `Silhouette`) — les ancres
    et libellés diabète restent définis ici ;
  - mappe `zones: SilhouetteZoneState[]` + `SILHOUETTE_ANCHORS` + labels vers les
    `SilhouetteZone[]` du générique et rend `<SilhouetteCorps …>` ;
  - re-exporte `SILHOUETTE_VIEWBOX` depuis le générique (une seule source de vérité).
  - `Silhouette.module.css` : vidé de ce qui a été déplacé ; le supprimer si plus rien ne l'importe.
- Interdit : renommer les états, changer une valeur d'ancre diabète, toucher un module diabète.
