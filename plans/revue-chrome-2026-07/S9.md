## D2 — Impression noir & blanc optimisée (cabinet)

### Objectif
Le livret reste lisible imprimé en N&B : pas d'information portée uniquement par la couleur.

### Décision clé
Palette douce (`oklch`) + blocs distingués par la couleur seule (« Bonnes pratiques » / « Erreurs »).
Étendre les `@media print` : bordures gris foncé/noir, fonds blancs, texte quasi-noir, eyebrows en gris
foncé, chips délimitées par bordure (pas seulement couleur de fond).

### Lire / Modifier
- Modifier : `src/components/PrintableLivret.module.css` (bloc `@media print`) ; compléments dans
  `src/styles/global.css` `@media print` si nécessaire

### Étapes

   libellé, épaisseur) en plus de la couleur.

## D2b — Pagination : couverture seule + flux compact + titres non orphelins

### Objectif
La couverture reste seule sur la 1ʳᵉ page ; ensuite flux compact (pas de saut forcé par section, pour
ne pas gaspiller de papier) ; cartes/items non coupés ; titres non orphelins.

### Décision clé
Règles déjà présentes à ajuster : `.cover { break-after: page }` (garder), `.section { break-inside:
avoid }` (aujourd'hui `auto` → passer `avoid`), `.sectionHead { break-after: avoid }` (garder). NB : la
page dédiée du **patch** est gérée en S13 (dépend de la dose), pas ici.

### Lire / Modifier
- Modifier : `src/components/PrintableLivret.module.css` (bloc `@media print`)

### Étapes
1. `.section` : `break-inside: auto` → `break-inside: avoid` (≈ l.407-409).
2. Vérifier `.cover break-after: page`, `.sectionHead break-after: avoid`, et les `break-inside: avoid`
   des cartes (substitut/outil/raison/bénéfice) → conservés.
3. Ne pas ajouter de `break-before: page` par section (flux compact voulu) — laisser `.sectionBreak`
   tel quel pour les sections qui le demandent explicitement.

### Validation
