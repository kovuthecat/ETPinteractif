## Résultat attendu

Remplacer la séquence codée actuelle (cellules-œufs) par l'animation illustration-driven à **4 modes**, contrôle **par cellule** :

| Mode | Cellules (c1,c2,c3) | Clés | Sang |
| --- | --- | --- | --- |
| Sans diabète | open, open, open | 3 | se vide → vert « bas » |
| Insulinopénie | open, closed, closed | 1 | ambre « encore élevé » |
| Insulinorésistance | rusty, rusty, rusty | 0 (plantées dans l'image) | rouge « élevé » |
| Mixte | rusty, open, closed | 1 (sur l'open) | rouge « élevé » |

## Travail

1. **Assets** : `cell-closed.png`, `cell-open.png`, `cell-rusty.png`, `key.png`, `sugar.png`, `pancreas.png` (recadré « en forme ») — transparents, produits par `build_assets.py` (les ajouter à sa table). Sources : cellules régénérées 2026-07-11 (fermée + `cellule ouverte.png` + `Cellule serrure rouillée.png`).
2. **Composant** : porter la logique du proto en React (état `mode`, config par mode = `{cell:[...], key:[...], sugar:n, txt, légendes}`), animation par **crossfade d'images empilées** (closed/open/rusty) + **clés en `transform` CSS** (pancréas → serrure centrale) + **artère sous les cellules** (jetons `sugar.png`, drain par mode, libellé tricolore vert/ambre/rouge). Reprendre les timings/positions du proto.
3. **Artère sous les cellules** : centrée sous les cellules, plus haute, jetons ~40 px (validé Thibault). **Option à tester (idée Thibault 2026-07-11)** : utiliser `artere-saine.png` comme **base de vaisseau** (au lieu de la barre rose codée du proto), pour la cohérence visuelle avec le M4 ; les jetons de sucre circulent dans la lumière. ⚠ L'artère saine est un segment compact — en format large sous 3 cellules elle risque de se déformer à l'étirement : prototyper les deux rendus (image-vaisseau étirée/tuilée vs barre codée) et **garder le plus lisible**. Le mécanisme de plaque croissante du M4 **ne s'applique pas** ici (M1 montre du sucre qui s'accumule, pas de plaque).
4. **Détails** : la cellule rouillée contient déjà une clé plantée → **pas de clé volante** en mode rouillé/mixte (sur la cellule rouillée). Resserrer le vide pancréas↔cellules (défaut noté au proto).
5. Respecter `prefers-reduced-motion` : proposer un état statique lisible (afficher directement l'état final du mode) si l'utilisateur a réduit les animations.
