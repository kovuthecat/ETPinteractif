## T3 — Afficher les illustrations des signes dans le bloc « Mes signes » de la carte-réflexe

### Objectif

Dans « Ma carte » (temps ③), remplacer les `<span class="chip">` texte-seul du bloc « Mes signes » par
le motif **illustration + label** déjà utilisé au temps ① « Mon profil hypo ». (Audit §8.)

### Décision clé

Le motif existe **dans le même fichier**, temps ① (l.207-216) :
```tsx
<div className={styles.preview}>
  {mySignes.map((s) => (
    <div key={s} className={styles.previewItem}>
      <IllustrationSlot id={`signe-${slugify(s)}`} label={s} size={100} />
      <span className={styles.previewLabel}>{s}</span>
    </div>
  ))}
</div>
```
`mySignes` (l.97) et `slugify` (l.70-77) sont déjà calculés et disponibles dans le composant.
Le bloc cible est temps ③, `.carteBloc` « Mes signes » (l.326-339), actuellement :
```tsx
<div className={styles.chipRow}>
  {mySignes.map((s) => (<span key={s} className="chip">{s}</span>))}
</div>
```

### Lire / Modifier

- Modifier : `HypoglycemieModule.tsx` (bloc temps ③, l.326-339) ; `HypoglycemieModule.module.css` si un
  format compact de `previewItem` est nécessaire pour tenir sur la carte.

### Étapes

1. Dans le `.carteBloc` « Mes signes » du temps ③ (l.326-339), remplacer le `.chipRow`/`<span class="chip">`
   par le motif `preview`/`previewItem` du temps ① (image + label). Garder le fallback « À définir avec
   le soignant » (l.336-338) quand `mySignes.length === 0`.
2. **Taille** : la carte-réflexe est plus dense que le profil. Utiliser `size={64}` (ou `72`) plutôt que
   `100`, pour que les N signes tiennent sans pousser la carte au scroll (à caler visuellement).
   Si besoin, introduire une variante CSS locale `.cartePreview`/`.cartePreviewItem` (flex, wrap, gap
   réduit) au lieu de réutiliser `.preview` tel quel — ne pas dégrader le rendu du temps ①.
3. Vérifier l'accessibilité : `IllustrationSlot` reçoit déjà `label` (nom accessible) ; le
   `<span className={styles.previewLabel}>{s}</span>` reste visible sous l'image.

### Validation

- Auto (bloque le commit) : `npx tsc --noEmit` → 0 erreur · `npm run build` → OK · `npm test` → vert.
- Humain (VALIDATION.md) : dans « Ma carte », les signes cochés au temps ① s'affichent **avec leur
  illustration** (comme dans « Mon profil hypo ») ; la carte reste lisible à ~1 m et **ne déborde pas**.
  La **fiche imprimée** (bouton « Imprimer ma carte ») garde ses chips texte.

### Si bloqué

Si les illustrations rendent la carte trop haute (scroll) même en `size` réduite → STOP, consigner la
hauteur observée, proposer un format en ligne (image + label côte à côte) et rendre la main.
