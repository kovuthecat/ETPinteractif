## Problème (capture #4)

« Remplacer le cercle blanc par l'image qui se trouve en son centre pour la rendre plus visible. » Le `.node` est un cercle crème de ~100 px (`border 3px`, `background: var(--color-surface)`, `box-shadow`) qui contient une image de seulement 44-56 px + un label + un hint → le **cercle blanc domine l'image**.

## Résultat attendu

Dans le rayonnement (centre + 4 rayons : Activité, Sucre, Cœur & vaisseaux, Tête, Autonomie), **l'illustration remplit le nœud**. Le cercle devient le cadre de l'image (plus de grand aplat crème vide). Le label passe **sous** le nœud, lisible.

## Travail

### 1. L'image occupe le nœud

- Faire en sorte que l'`IllustrationSlot` du nœud (`activite-centre`, `activite-rayon-*`) fasse **quasiment la taille du nœud** (image ronde plein cadre), au lieu de flotter au centre d'un disque crème.
  - Option A (recommandée) : le nœud lui-même devient le cadre image — fond transparent, l'image (`shape="circle"`, `object-fit: cover`) remplit `.nodeButton` (100 %). Retirer/atténuer le `background: var(--color-surface)` du `.node` (il ne doit plus rester un grand aplat blanc visible).
  - Passer un `size` d'image cohérent avec le rendu du nœud (le nœud fait `NODE_R_PCT*2 = 21%` d'un conteneur ≤ 480 px ≈ 100 px ; centre `CENTER_R_PCT*2 = 26%` ≈ 125 px). Viser une image qui remplit ce diamètre.
- Conserver l'anneau de bordure comme **état** (repos `--color-line`, allumé `--color-confort` via `.nodeLit`, sucre `--color-vigilance` via `.nodeSucre`) — c'est l'anneau, pas un disque plein, qui porte l'état.

### 2. Label sous le nœud, hint retiré

- Sortir le `.nodeLabel` **sous** le cercle (pas par-dessus l'image) pour qu'il reste lisible.
- Retirer le hint intra-nœud (`.nodeHint` « cliquez : un effort, tous les bénéfices » et le `.nodeSuite` « voir la suite ») **de l'intérieur du cercle** : soit sous le label, soit supprimé (cf. passe texte S8 — ici on le sort au minimum de l'image). Le badge info `.nodeInfo` (i) reste, posé en coin.

### 3. Vérifier lisibilité de l'allumage

- Quand on clique le centre (« tous les bénéfices ») ou un rayon, l'état allumé (`.nodeLit` : bordure + `--color-confort-soft`) doit rester visible **malgré l'image plein cadre** (l'anneau doit ressortir). Ajuster l'épaisseur/contraste de l'anneau si l'image « mange » la bordure.
