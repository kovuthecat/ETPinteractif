## T3 — Remplacer les lettres « R »/« A » par des icônes Lucide

### Objectif

Sur les graphes de glycémie, les marqueurs affichent une **lettre** (« R » = repas, « A » = injection d'insuline).
Les remplacer par des icônes Lucide plus lisibles.

- **R (Repas) → `Utensils`**
- **A (injection d'insuline) → `Syringe`** *(confirmé : « A » = moment de l'injection)*

{meta.glyph}</text>` (l.356-357), centrée dans le `<circle r={12}>` (`.marqueurBadge`).

### Lire / Modifier

1. Ajouter à chaque entrée de `MARQUEUR_META` un champ icône (composant Lucide) : `Utensils` pour le type
   « repas », `Syringe` pour le type « injection/action ».
