## T1 — Auto-héberger les polices

### Objectif
Intégrer Source Serif 4 + Work Sans en local (hors-ligne), sans Google Fonts.

### Lire / Modifier
- Copier depuis `maquettes/reference/fonts/` → **créer** `public/fonts/` avec les 4 fichiers
  (`SourceSerif4-latin.woff2`, `SourceSerif4-latinExt.woff2`, `WorkSans-latin.woff2`, `WorkSans-latinExt.woff2`).
- Modifier : `src/styles/global.css` (blocs `@font-face`), éventuellement `index.html` (préchargement).

### Étapes
1. Copier les 4 woff2 dans `public/fonts/` (servis à `/fonts/…` par Vite).
2. Dans `global.css`, écrire 2 `@font-face` par famille (latin + latin-ext), `font-display:swap`,
   `src: url('/fonts/<fichier>') format('woff2')`, avec les **mêmes `unicode-range`** que la maquette
   (copier depuis `snapshot.html`). Poids : déclarer les familles comme variables (une face couvre les poids)
   — reprendre la structure de `snapshot.html` (`font-weight` en plage ou faces multiples pointant le même fichier).
3. (Option) `index.html` : `<link rel="preload" as="font" type="font/woff2" href="/fonts/WorkSans-latin.woff2" crossorigin>`
   pour le texte principal. Pas de `<link>` vers un domaine externe.

### Validation
- Auto : `vite build` → succès ; les woff2 sont copiés dans `dist/fonts/`.
- Humain : titres en serif, UI en sans, accents FR (é è à ê) nets → `VALIDATION.md`.

### Si bloqué
