## Décision clé — modèle cible (transcrit du handoff, autoritaire)

Le modèle a **changé d'échelle et de forme** (avant : 0–1 abstrait, cigarette/patch/ponctuel/vapoteuse ;
maintenant : **0–100 sur 24 h**, dose par événement, types cigarette/patch/substitut, tension découplée).

**Nicotine — constantes & noyaux** (`t` en heures) :
```
TIME_MAX = 24 ; LEVEL_MAX = 100 ; BASELINE = 4
type NicotineEventType = 'cigarette' | 'patch' | 'substitut'
type NicotineEvent = { type: NicotineEventType; time: number; dose?: number }  // time ∈ [0,24] ; dose (patch) en multiples de 0.25, défaut 1
cigarette(dt) = dt<0 ? 0 : 40 * (1 - exp(-dt/0.04)) * exp(-dt*LN2/1.2)
patch(dt,dose) = dt<0 ? 0 : (dt<0.5 ? 30*dose*(dt/0.5) : 30*dose)      // rampe 0.5 h puis plateau persistant
substitut(dt) = dt<0 ? 0 : 26 * (1 - exp(-dt/0.35)) * exp(-dt*LN2/2.2)
niveau(t, events) = clamp( BASELINE + Σ contribution(ev, t) , 0, LEVEL_MAX )
```
- Échantillonnage : 200 pas (`i/200*TIME_MAX`) comme le handoff (ou n paramétrable, défaut 200).
- **Zones** (bandes MANQUE / CONFORT / SURDOSAGE) : lire les bornes exactes dans le `.dc.html` (rects de bande).
  Repère de rendu du handoff : `levelToY(l) = 220 − (l/100)*200` ; labels aux niveaux ≈ MANQUE bas (~4), CONFORT médian (~45),
  SURDOSAGE haut (~90). Fournir un `classifyZone(level)` avec les 2 seuils lus dans le handoff.

**Tension (soulagement) — découplée de la nicotine** :
```
TENSION_HIGH = 90 ; TENSION_TROUGH = 15 ; TENSION_NONSMOKER = 4 ; TENSION_TAU = 2.2 ; VIRTUAL_START = -2.5
tensionLevelAt(t, cigTimes) :
  lastT = max({ VIRTUAL_START } ∪ { c ∈ cigTimes | c ≤ t })
  return TENSION_HIGH - (TENSION_HIGH - TENSION_TROUGH) * exp(-(t - lastT)/TENSION_TAU)
// non-fumeur = constante TENSION_NONSMOKER (ligne repère « Comparer au non-fumeur »)
```
Note : la tension ne dépend **que des instants de cigarette** (plus du niveau de nicotine comme avant).

**Invariants à couvrir par tests** :
- [ ] niveau toujours ∈ [0,100] ; sans événement = BASELINE constant (4).
- [ ] 1 cigarette : pic ≈ juste après `t0`, puis décroissance (demi-vie ~1.2 h) ; substitut plus bas/plus lent que cigarette.
- [ ] patch dose 1 : plateau ≈ 30 après 0.5 h ; dose 2 : plateau ≈ 60.
- [ ] tension : au moment d'une cigarette → proche de TROUGH (15) ; longtemps après → tend vers HIGH (90) ; non-fumeur constant 4.
- [ ] repère non-fumeur (4) toujours < creux fumeur (min 15) — invariant « le fumeur ne fait que revenir au niveau du non-fumeur » conservé.
