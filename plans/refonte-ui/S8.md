## T10 — Aligner le module Craving sur le handoff

### Objectif
Reproduire la mécanique du handoff : une **vague de 3 minutes** qu'on laisse passer, avec 4 techniques (4D) activables.

### Décision clé (handoff)
- **Durée = `CRAVING_DURATION = 180 s`** (pas ~30 s). Phases `idle → active → done`. Horloge en compte à rebours
  `formatClock` (**3:00 → 0:00**). « Une envie arrive ? » (idle) → « La vague est en train de redescendre » + horloge (active) → « Elle reviendra peut-être, un peu plus tard ou plus faible… » (done). Bouton pour (re)lancer.
- **Vague** = 3 segments de **Bézier cubique** (`CRAVING_WAVE_SEGMENTS`), tracée progressivement (`cravingWavePathUpTo(progress)`,
  `progress = 1 − cravingTime/180`) ; **marqueur « maintenant »** (`cravingDotX/Y = cravingWavePoint(progress)`) qui parcourt la vague.
- **4 D** (`CRAVING_D_INFO`, multi-sélection `cravingDs[]`, chip « ✓ <titre> » quand actif) :
  `differer` **« Différer »** · `detourner` **« Détourner l'attention »** · `detendre` **« Se détendre — respirez »** · `eau` **« D'eau »**.
  Chaque carte a sa couleur/`colorSoft` et son `body` (textes du handoff). **« Se détendre »** = animation de respiration (cercle
  `@keyframes etp-breathe`, inspire en grandissant / expire en se resserrant), active tant que la carte est activée.
- Aparté « Tabac Info Service — 39 89 » (déjà dans le bloc Sources / bas de module).

### Étapes
1. État : `cravingPhase`, `cravingTime` (init 180), `cravingDs: string[]`. `start`/`reset` gèrent l'`interval` (1 s), cleanup à l'unmount.
2. SVG vague : path via `cravingWavePathUpTo(progress)`, marqueur « maintenant » à `cravingWavePoint(progress)` ; libellés d'état + horloge selon la phase.
3. 4 cartes D = `.card` (couleur/soft du D), toggle multi-sélection, chip « ✓ » quand actif ; « Se détendre » anime le cercle de respiration.
4. Boutons = primitives `.btn` ; cibles ≥44 px. Reprendre tailles/couleurs exactes du `.dc.html`.

### Validation
- Auto : `tsc -b` + `vite build` → succès.
- Humain (→ `VALIDATION.md`) : compte à rebours 3:00→0:00, marqueur qui parcourt la vague, phases idle/active/done, 4 D aux **bons libellés** (Différer / Détourner l'attention / Se détendre — respirez / D'eau) multi-activables, respiration anime, timer nettoyé.
