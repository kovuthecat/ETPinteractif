# S8 — Hypoglycémie : afficher toutes les illustrations des signes sélectionnés

> **Mode** : solo · **Exécutant** : Sonnet · **Effort** : xhigh · **Dépend de** : S1 (chrome élargi).
> **À lire avant** : index v3 §2 · `src/features/diabete/hypoglycemie/HypoglycemieModule.tsx` + `.module.css`.

## Problème (audit 8) — bug

Onglet « Mon profil hypo », section « Mes signes précoces » : quand plusieurs signes sont sélectionnés (Sueurs, Palpitations, Faim soudaine, Irritabilité), le panneau `.preview` n'affiche que **l'illustration du dernier signe cliqué**.

**Cause** : la sélection multiple **existe déjà** (`signes: Record<string,boolean>` ~l.88 ; `mySignes = SIGNES.filter(s => signes[s])` ~l.99 contient **tous** les signes). Mais un état séparé stocke seulement le dernier clic :
```ts
const [lastSigneClicked, setLastSigneClicked] = useState<string | null>(null); // ~l.89
```
et le preview ne lit QUE lui (~l.202-207) → une seule illustration. Effet de bord : désélectionner un signe ne vide pas `lastSigneClicked` → une illustration d'un signe **non** sélectionné peut rester affichée.

## Résultat attendu

Le preview affiche **toutes** les illustrations des signes actuellement sélectionnés (et rien si aucun).

## Travail

1. Remplacer le bloc `~l.202-207` par un rendu de **tous** les `mySignes` :
```tsx
{mySignes.length > 0 && (
  <div className={styles.preview}>
    {mySignes.map((s) => (
      <div key={s} className={styles.previewItem}>
        <IllustrationSlot id={`signe-${slugify(s)}`} label={s} size={100} />
        <span className={styles.previewLabel}>{s}</span>
      </div>
    ))}
  </div>
)}
```
2. Supprimer l'état mort : `lastSigneClicked` (~l.89) et l'appel `setLastSigneClicked(signe)` dans `toggleSigne` (~l.96 → la fonction se réduit à `setSignes(prev => ({ ...prev, [signe]: !prev[signe] }))`).
3. `.module.css` — `.preview` (~l.68-76) est prévu pour un seul item : le passer en `flex-wrap` + introduire `.previewItem` :
```css
.preview { display:flex; flex-wrap:wrap; align-items:center; gap:var(--space-md);
  padding:var(--space-md) var(--space-lg); background:var(--color-surface);
  border:1.5px solid var(--color-line); border-radius:var(--radius-lg); }
.previewItem { display:flex; flex-direction:column; align-items:center; gap:var(--space-sm); width:110px; }
```
(passer `.previewLabel` en `text-align:center` si besoin.)
