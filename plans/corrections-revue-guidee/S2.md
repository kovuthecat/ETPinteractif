## T1-B — Rendre la technique de prise (vapoteuse) visible sans défiler

### Objectif
Sur la forme **Vapoteuse**, le bloc `TECHNIQUE DE PRISE — VAPOTEUSE` (illustration main+vapoteuse+horloge, info
clé d'usage) est **sous la ligne de flottaison** : il faut scroller pour le voir. Le rendre visible **sans
défiler** à la sélection de la forme (point 6).

### Décision clé
Ce n'est **pas** un débordement (rendu correct dans le cadre) mais une **priorité de contenu** : trop de hauteur
au-dessus. Trois leviers possibles (choisir le plus simple qui suffit, à valider à l'œil) :
1. **Compacter le bloc supérieur** (réduire marges/paddings/hauteur des sections au-dessus de la technique) ;
2. **Réordonner** : remonter la carte technique juste sous les onglets de forme ;
3. **Scroll programmatique** à la sélection de la forme (`ref` sur la carte technique + `scrollIntoView({block:
   'nearest'})` dans l'effet de changement de forme) — filet si (1)/(2) ne suffisent pas.
Préférer (1) puis (2) ; ne recourir à (3) que si nécessaire (pas d'animation agressive ; respecter
`prefers-reduced-motion`).

### Lire / Modifier
- Modifier : `SubstitutsModule.module.css` (compactage/ordre) et/ou `SubstitutsModule.tsx` (ordre des blocs
  ou `scrollIntoView` ciblé sur la carte technique).

### Étapes
1. Repérer, pour la forme Vapoteuse, la pile de blocs entre les onglets de forme et la carte technique.
2. Appliquer (1) puis, si insuffisant, (2) ; en dernier recours (3).
3. Vérifier les **6 formes** (Gomme/Pastille/Comprimé/Spray/Patch/Vapoteuse) : le compactage/réordonnancement ne
   doit dégrader aucune autre forme.

### Validation
- Auto (bloque le commit) : `npx tsc --noEmit` · `npm run build` · `npm test` → verts.
- Humain (visuel) : sur Vapoteuse, l'illustration de technique de prise est visible **sans scroller** (viewport
  desktop standard) ; les 5 autres formes restent correctes. → `VALIDATION.md`.

### Si bloqué
Si aucune combinaison (1)/(2) ne fait entrer la technique dans le viewport sans casser une autre forme → poser (3)
et signaler ; si (3) elle-même est fragile → STOP, décrire la structure.
